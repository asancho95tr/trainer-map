# 🃏 ¿Cómo funcionan los Arcanos Menores?

Se dividen en **4 palos**, muy parecidos a una baraja normal:

| Palo       | Equivalente | Tema principal             |
| ---------- | ----------- | -------------------------- |
| Copas 🏆   | Corazones   | Emociones, amor            |
| Oros 🪙    | Diamantes   | Dinero, trabajo            |
| Espadas ⚔️ | Picas       | Conflictos, mente          |
| Bastos 🔥  | Tréboles    | Acción, energía, proyectos |

---

## 📊 Cada palo tiene 14 cartas

* **As (1)**
* **2–10**
* **Sota**
* **Caballero**
* **Reina**
* **Rey**

👉 14 × 4 = **56 cartas**

---

# 🔮 Cómo se interpretan (simplificado para app)

## 🧠 1. Número = tipo de energía

| Número | Significado general |
| ------ | ------------------- |
| As     | Inicio              |
| 2      | Dualidad/decisión   |
| 3      | Crecimiento         |
| 4      | Estabilidad         |
| 5      | Conflicto           |
| 6      | Armonía             |
| 7      | Desafío             |
| 8      | Progreso            |
| 9      | Culminación         |
| 10     | Cierre              |

---

## 🎭 2. Palo = contexto

Ejemplo:

* **3 de Copas** → celebración emocional 🥂
* **5 de Espadas** → conflicto mental ⚔️
* **10 de Oros** → éxito económico 💰

---

## 🔄 3. Invertidas

Igual que las mayores:

* Bloqueo
* Exceso
* Energía negativa o interna
